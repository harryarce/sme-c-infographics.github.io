# SME&C Infographics

[View current Infographics](https://sme-c-infographics.github.io/)

A single destination for infographics that help visualize and understand our presentations and Azure systems. Browse the live site to find quick-reference visuals across Microsoft's core cloud services and solutions.

## How to Contribute

1. [Creating HTML Infographics with CoWork](docs/creating-html-infographics-with-cowork.md)
2. [Adding an Infographic to the Website](docs/adding-an-infographic-to-the-website.md)

## Automation

Site chrome (analytics, back button, meta tags, favicon, manifest) is
applied automatically by GitHub Actions on every PR and post-merge.
See [docs/automated-page-updates.md](docs/automated-page-updates.md)
for the full automation design — audit scripts, link/terminology
checks, accessibility scan, and the monthly LLM-assisted content
freshness report.
