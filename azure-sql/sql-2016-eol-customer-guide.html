What Does End of Support Mean?

After the end-of-support date, Microsoft will no longer provide security updates for SQL Server 2016—leaving your databases exposed to unpatched vulnerabilities.

FAQ

Frequently Asked Questions

What happens after July 14, 2026? ▼
Microsoft stops providing security updates for SQL Server 2016. No more critical or important patches, leaving databases vulnerable to newly discovered threats and potentially out of compliance.
Are ESUs free on Azure VMs for SQL 2016? ▼
No. Unlike SQL Server 2014, ESU for SQL Server 2016 is paid across all environments—including Azure VMs, Azure Local, and Azure VMware Solution. This creates consistent pricing across all channels.
What is the Savings Plan for Databases? ▼
A flexible way to save up to 35% by committing to a fixed hourly dollar amount on database services for 1 year. Savings apply globally across participating services. It's ideal for dynamic workloads where usage varies. For stable workloads, Azure Reservations can save up to 55%.
What is Azure Hybrid Benefit? ▼
Azure Hybrid Benefit lets you use your existing on-premises SQL Server licenses with Software Assurance in Azure, saving up to 55%. Azure is the only cloud that allows you to bring your SQL license to PaaS (Managed Instance). Combine with Reserved Capacity for savings up to 80%.
Can I use Azure Arc for ESU if I'm not in Azure? ▼
Yes! Azure Arc lets you enroll and purchase ESUs directly in the Azure Portal for on-premises and multi-cloud servers. You get pay-as-you-go billing, centralized visibility, and can stop charges immediately when no longer needed.
What is Next-Gen General Purpose? ▼
Next-Gen GP is a new Azure SQL Managed Instance tier that uses premium managed disks instead of page blobs. It delivers 3–5x better performance for OLTP and analytical workloads, 5x better database density, 2x storage scale, and 2x improved vCore flexibility—all at the General Purpose price point.
